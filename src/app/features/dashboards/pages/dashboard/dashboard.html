Sobre o projeto

App em Angular 20 consumindo a API pública do Rick and Morty, com páginas para personagens, episódios e locais, busca, paginação infinita e rotas protegidas.

  • Angular 20, TypeScript e RxJS 7
  • Roteamento por features e lazy loading
  • Infinte scroll e busca (personagens)
  • Guarda de rota para página de perfil
  • Estilo com Bootstrap 5
