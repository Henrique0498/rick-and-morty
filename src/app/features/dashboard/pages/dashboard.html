Projeto

Este projeto foi desenvolvido utilizando Angular, consumindo a API pública do Rick and Morty. O objetivo é fornecer uma interface interativa para explorar os personagens, episódios e locais da série animada.

Tecnologias Utilizadas

  • Angular
  • TypeScript
  • Bootstrap

API Rick and Morty

A API do Rick and Morty é uma API RESTful que fornece informações detalhadas sobre os personagens, episódios e locais da série. Ela é de uso gratuito e pode ser acessada em rickandmortyapi.com.
